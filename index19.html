    3. Target Attribute

      4. The target attribute used many values like _blank, _self, _parent, _top






    4. Autocomplete Attribute

      5. The browser automatically complete values based on values that the user has entered before.






    5. Novalidate Attribute

      6. Adding novalidate to the form tag prevents this validation, allowing submission even if the fields are invalid.







  2. HTML Form Elements

    1. Input and label element

      • The input element is used for get the input from user.
      • The label element is useful for screen reader users.




    2. Select element

      • Selection element defines the drop down list.
      • The "option" element defines an option that can be selected.
      • Default first item in drop-down list is selected, but pre-selected option also there, using "selected" attribute
      • Use the "size" attribute to specify the number of visible values.
      • Use the "multiple" attribute to allow the user to select more than one value






    3. Textarea element

      4. The "textarea" element defines the multi-line input field.



    4. Button element

      5. The "button" element defines the clickable button.



    5. fieldset and legend element

      6. The "fieldset" element is used to group related data in a form.

      The "legend" element defines a caption for the fieldset.

    6. datalist element

      7. The "datalist" element specify a list of pre-defined options for a input elements.

      It look same as "autocomplete" attribute.
