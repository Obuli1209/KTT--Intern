Javascript

Javascript is a interpreted programming language, event driven, high-level.

JS use JIT->just-in-time


Components

  • Parser-js code into AST(Abstract syntax Tree)
  • Interpreter-read and execute AST line by line.
  • JIT Compiler-execute code into machine code.
  • Garbage collector
  • Chrome-Interpreter(Ignition), Garbage collector(Orinoco)

Versions

  • 1995-Brendan Eich-Netscape-Mocha-> Livescript->javascript
  • 1996-Microsoft introduced js in Internet Explorer.
  • 1997-ECMA introduce ES1
  • 1998-ES2-updates
  • 1999-ES3-trycatch
  • 2009-ES5-JSON support
  • present-ES14

Code Editors

  • VS Code
  • Atom
  • Webstorm
  • Notepad++

Keywords

  • var-> function scope, used in outside of block also, redeclare in same scope.
  • let-> block scope, used only in inside of block, cannot redeclare in same scope.
  • const-> block scope, once declare didnot change again, used only in inside of block, cannot redeclare in same scope.

Datatypes

  • Number
  • String
  • Boolean
  • Bigint
  • Object
  • Undefined
  • null

Mutable and Immutable

  • Immutable-> not changeable, primitive datatype, stack memory - stored by value.

    Immutable means assign a new value, creates a new memory allocation rather than modifying the existing one.

    Number, string, boolean, bigint.

  • Muttable-> changeable, non-primitive datatype, heap memory- stored by reference.

    modify the value in exisiting one.

    Object,arrays,functions.

    Object.freeze()
