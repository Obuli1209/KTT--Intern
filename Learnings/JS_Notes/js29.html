JavaScript has 8 Datatypes

JavaScript has dynamic datatypes. This means that the same variable can be used to hold different data types:

  1. Primitive datatypes - 7

    2. Immutable means assign a new value, creates a new memory allocation rather than modifying the existing one.

    Stored the value in stack memory

    • Number
    • String
    • Bigint
    • Boolean
    • undefined
    • null
    • symbol

  2. Non-primitive datatype - 1

    3. It is muttable. It modify the value in exisiting one.

    Stored the value in heap memory by reference.

    • Object - A collection of key-value pairs.
    • Arrays
    • Functions

Nullish Coalescing operator

?? is useful for providing default value, dealing with null or undefined values.

let a = 0;
console.log(a || "Good day");
// ( Logical OR || check all falsy values.)

Usecase- check falsy values like (false, 0, "", null, undefined, NaN.)

console.log(a ?? "Good day");
// (Nullish ?? check for null or undefined.)

Usecase- check only null or undefined, keeping (false, 0, "")
